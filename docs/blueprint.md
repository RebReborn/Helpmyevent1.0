# **App Name**: HelpMyEvent

## Core Features:

- Event Posting: Create and post detailed event information including type, date, location, service needs, and inspirational images.
- Service Provider Profiles: Enable service providers to create profiles showcasing their skills, portfolios, experience levels, and availability for free work.
- Offers & Collaboration: Facilitate communication and collaboration through messaging, offer submissions, and mutual agreement pages between event posters and providers.
- Portfolio & Reviews: Allow both event posters and service providers to leave reviews and showcase project outcomes, building a portfolio and testimonial system.
- Event Discovery: Provide a calendar and search functionality allowing users to filter events by date, category, and location with a map view of nearby opportunities.
- AI-Powered Recommendation Tool: AI tool suggesting relevant service providers based on event details and desired services using matching algorithms and machine learning.
- Social Networking: Build a social layer by implementing 'follow', 'like', and 'comment' functionalities, including 'Featured Collaborations' and community blog.

## Style Guidelines:

- Primary color: A vibrant, energetic purple (#9400D3) to represent creativity and collaboration.
- Background color: A light lavender (#E6E6FA) to provide a soft and clean backdrop.
- Accent color: A lively magenta (#FF00FF) to highlight calls to action and important elements.
- Headline font: 'Belleza' sans-serif font for a personable, arts-forward, modern aesthetic.
- Body font: 'Alegreya' serif font, to create readability where blocks of text need to appear.
- Use clear, modern icons to represent different service categories (e.g., photography, DJ, catering) and event types (e.g., wedding, birthday).
- Implement a clean, grid-based layout to showcase events and provider profiles. Use white space effectively to avoid clutter.
- Use subtle animations and transitions to enhance user experience when loading content or navigating between sections.
